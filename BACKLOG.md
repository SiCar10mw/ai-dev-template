# Backlog

Human curates this board. The backlog worker claims exactly one item, moves it to `🔨 In Progress`, implements it through the spec + test + PR conventions, moves it to `👀 Review`, and stops for human approval.

The operating principle is: **manage the work, don't supervise the agent**. The human sets priorities and approves outcomes. The worker runs the selected item end-to-end without self-merging.
