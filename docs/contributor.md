# Contributor Guide

## Start Here

1. Read `CONSTITUTION.md`.
2. Read `STANDARDS.md`.
3. Run `make check`.
4. For feature work, start under `specs/`.

## Local Commands

```bash
make check
pytest
ruff check .
```

## Review Expectations

Pull requests should include tests, docs updates when behavior changes, and enough evidence for an independent reviewer to reproduce the result.
